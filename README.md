## Key bindings

| Key | Function |
| :--- | :--- |
| < f2 > | magit-status |
| < f7 > | Create new empty buffer |
| < f8 > | ispell-word |
| M-< f8 > | flyspell-check-next-highlighted-word |
| < f9 > | Toggle sroll bar  |
| C-c C-c | copy-line |
| C-c left  | Winner mode - Undo windows changes  |
| C-c right  | Winner mode - Redo windows changes  |
